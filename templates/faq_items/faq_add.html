How to Add Quotes

Add Command Format

/add <user> <quote> [<context>]

  • /add :
    • The command name, type this into a channel on Discord to start adding a quote.

  • <user> :
    • The discord user you wish to add a quote to.

  • <quote>:
    • The text content of the quote you want to add.

  • <context> : (optional)
    • Write a short entry to give extra context to the quote.
